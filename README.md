BSA2014-NodeJS
===============================

## Hometask boilerplate for JS group after NodeJS lection.

### How to start the server
- Install the [node.js](http://nodejs.org/)
- Clone this repository
- Go to folder
```shell
cd BSA2014-NodeJS/
```
- Install all the dependencies
```shell
npm install
bower install
```
- Run the server
```shell
node app.js
```
- Go to browser and check the API.
```
localhost:3000
```
